# Setup Directory

This directory contains the setup wizard configuration files.

## Files

- **wizard-prompts.md** - Prompt templates for each configuration question
- **config-schema.json** - JSON Schema for validating user configuration
- **preflight-checks.md** - Pre-flight check instructions for Claude

## Usage

These files are used by Claude when running the interactive setup wizard. They define the conversation flow, validation rules, and system requirements.
